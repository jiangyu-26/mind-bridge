# Mind Bridge - 心理 AI 助手

基于 Vue 3 + Vite 开发的心理健康 AI 咨询平台。

## 技术栈

- Vue 3 + Composition API + Vite
- Element Plus UI 组件库
- Vue Router + Pinia
- Axios + 请求拦截器
- ECharts 数据可视化
- SSE 流式通信

## 功能模块

### 用户端
- AI 心理咨询（流式对话）
- 情绪日记记录与分析
- 心理知识库浏览

### 管理端
- 数据看板
- 知识文章管理
- 咨询记录管理
- 情绪日志管理

## 运行

```bash
npm install
npm run dev
```
